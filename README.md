## Devices (Tested)

* iPhone11 - (iOS 15.5)

## Stack Requirements (Tested)
- Node (14.20.0)
- Expo-cli (6.0.5)
- Expo SDK (47.0.6)
- React native (0.70.5)


## Run project

1. Install dependencies

```bash
yarn install
expo install
```

2. Start project


```bash
expo start
```
